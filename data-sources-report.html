Detailed Source Investigations

CZ SZP CR — Czech Reimbursement Prices IMPORTABLE

Ready for import — structured XLS, freely downloadable

Source URL: szpcr.cz/zdravotnicke_prostredky
Format: XLS with 25 structured columns
Coverage: All Czech health insurers, surgical implants (groups 01, 41–92)
Price scope: component (per-device unit prices in CZK)
Key fields: KOD, NAZ (name), DOP (desc), VYR (manufacturer), AKC (price), CMF (regulated price), MAX (max reimbursement), UHP (%), JKP (classification)
Currency: CZK → EUR at ECB rate (~25.3 CZK/EUR)
EMDN codes: Mandatory since Dec 2025 — direct category linking
Import script: npm run import:cz-szpcr
Estimated entries: 500–2,000+ orthopedic implant rows
Coverage impact: +10–20% product match rate

Supersedes VZP ZUM PDF — same underlying data in structured XLS format, eliminating the need for PDF/OCR extraction.

GB NHS Tariff — UK Procedure Prices IMPORTABLE

Ready for import — XLSX workbook, freely downloadable

Source: NHS England National Payment Scheme workbook (XLSX), published annually
Format: XLSX with structured columns per HRG code
Coverage: HRG procedure tariffs for orthopedic procedures
Price scope: procedure (total cost incl. implant + surgery)
Relevant HRG codes: HB11–HB22 (joint replacements), HB31–HB39 (revisions)
Currency: GBP → EUR at ECB rate
Import script: npm run import:gb-nhs
Estimated entries: ~25 orthopedic HRG codes
Coverage impact: +1% product matches, adds UK as reference geography

Limitation: No per-implant breakdown — implant cost is ~15–20% of total procedure tariff.

EU TED Open Data NOT IMPORTABLE

Investigated, not useful for per-device pricing

Access: CSV bulk files at data.europa.eu, API at api.ted.europa.eu
Fields: Notice ID, buyer, CPV code, award value (EUR), winner, lot number, country

Only lot/contract-level totals (e.g., "EUR 2M for 4-year hip implant framework"), no unit prices or device-level breakdown. CPV 33183100 (orthopedic implants) is too broad. Supplier names are free-text and inconsistent.

Use case: Market structure analysis only (which suppliers win in which countries), not reference pricing.

GB NHS Supply Chain / TOS3 Framework BLOCKED

Commercially confidential, behind authentication

Framework: TOS3 (Feb 2024 – Jan 2028), covers all orthopedic implants with banded volume pricing
Access: NHS trusts only via my.supplychain.nhs.uk

NJR EMBED/INFORM benchmarking tools aggregate trust-level prices but access restricted to subscribers. No public CSV/Excel/API export.

Not accessible without NHS procurement contact.

IN NPPA — Indian Ceiling Prices NOT IMPORTED

Technically trivial but low value for EU-focused catalog

Coverage: 19 entries — 15 primary + 4 revision knee components only
Price level: 70–85% below EU (Rs 51,563 ≈ EUR 543 for TKR vs EUR 2,000–8,000 in EU)
Limitation: No manufacturer dimension, INR/EUR volatile (16% swing in 2025)
Format: Government Gazette PDF only

Skipped: Non-EU, minimal coverage, price levels not comparable.

CZ openprocurements / Hlidac statu NOT IMPORTABLE

Only lot-level contract values, no unit prices

Data: 60+ orthopedic tenders via CPV 33183100, lot-level estimated values
Unit prices: In PDF attachments on TenderArena (behind auth)

Same fundamental limitation as TED: no per-device price breakdown.

NJR / ODEP (UK) CATALOG ENRICHMENT ONLY

No pricing data, useful for product catalog enrichment only

NJR Annual Reports: Top implant brands by usage volume, revision rates — CSV from reports.njrcentre.org.uk
ODEP Ratings: Quality ratings for 200+ implant brands (10A*, 13A*, etc.) at odep.org.uk

Useful for brand validation and quality metadata, not for reference pricing.

SUKL ZPSCAU (CZ) NOT USEFUL

12,848-row XLSX inspected (March 2024). Contains only voucher-dispensed outpatient devices (braces, hearing aids, wound care). Zero surgical implant entries — orthopedic implants reimbursed through hospital DRG.

SK SZM Details

SK MZ SR SZM — Slovak Reimbursement Registry

Source file: health.gov.sk/.../Zoznam_SZM_YYYYMM.xlsx
Update cadence: Quarterly (Jan 1, Apr 1, Jul 1, Oct 1)
XC code groups: XC1 (hip), XC2 (knee), XC3 (other joints), XC4 (osteosynthesis), XC5 (arthroscopy)
Price columns: UZP (per-device reimbursement), UZP2 (category-level ceiling per set)
XC-to-EMDN mapping: AI-assisted via Gemini, cached in scripts/data/sk-xc-emdn-mapping.json
